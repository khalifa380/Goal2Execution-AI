# Goal2Execution-AI

Goal2Execution-AI is an Agentic AI platform that transforms plain-English instructions into autonomous multi-step workflows. The system uses Large Language Models (Gemini and Claude) with LangChain to plan tasks, select tools, execute actions, and generate final outputs without hardcoded workflows.

The platform includes web search, calculator, and file-writing tools, a recommendation engine built using TF-IDF and cosine similarity, a Flask web interface, and automated testing and CI/CD pipelines using pytest, flake8, and GitHub Actions.
